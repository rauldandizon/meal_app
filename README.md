# Meals App

A Flutter project that demonstrates a simple meals recipe application.

## Description

This app allows users to browse through a variety of meal categories, view the recipes for each meal, and save their favorite meals. It also includes a filtering feature to help users find meals that fit their dietary needs (e.g., gluten-free, vegetarian).

## Features

-   **Browse Meal Categories:** View a list of meal categories like Italian, German, etc.
-   **View Meals:** See a list of meals for each category.
-   **Meal Details:** Get detailed information about a specific meal, including ingredients and steps to prepare it.
-   **Favorites:** Mark meals as favorites and view them in a separate "Favorites" screen.
-   **Filtering:** Filter meals based on dietary preferences like:
    -   Gluten-free
    -   Lactose-free
    -   Vegetarian
    -   Vegan
-   **State Management:** Utilizes `flutter_riverpod` for efficient and scalable state management.

## Dependencies

-   `flutter_riverpod`
-   `google_fonts`
